# Syntecxhub_Project_ExpenseTrackerCLI

Expense Tracker CLI (Python)

📌 Project Overview

The Expense Tracker CLI is a command-line application built using Python that helps users record and manage their income and expenses efficiently. The project demonstrates practical implementation of: File handling (CSV) Date handling Data aggregation Monthly summaries Data export (Excel/CSV) Basic data visualization using Matplotlib This project was developed as part of an internship task.

🚀 Features

✔ Add income and expense entries

✔ Store data persistently in a CSV file

✔ View monthly financial summary

✔ Export data to Excel/CSV

✔ Generate simple charts (PNG image files)

✔ Command-line interface (CLI based)

🛠 Technologies Used

.Python 3 CSV module (File I/O) Datetime module Pandas (for Excel export) Matplotlib (for chart generation)

📂 Project Structure

expense_tracker.py

expenses.csv (auto-generated)

monthly_summary.csv (generated)

expense_chart.png (generated)

📊 Functionalities

1️⃣ Add Transaction Enter Date (YYYY-MM-DD) Enter Category (Food, Travel, Salary, etc.) Enter Amount Select Type (Income / Expense)

2️⃣ Monthly Summary Calculates total income Calculates total expenses Displays net balance

3️⃣ Export Options Export summary to CSV Export data to Excel file (.xlsx) Generate monthly expense chart (PNG image)

🎯 Learning Outcomes

This project demonstrates:

CLI-based application development

Working with CSV files

Date filtering and monthly aggregation

Basic data visualization

Clean and structured Python coding practices
